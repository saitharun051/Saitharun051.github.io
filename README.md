# PhD Research Portfolio

Personal research portfolio with AI-powered agent for answering questions about my research, projects, and publications.

## Tech Stack
- React + Vite
- Tailwind CSS
- Google Gemini API
- Cloudflare Workers

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Automatically deploys to GitHub Pages via GitHub Actions on push to main.
